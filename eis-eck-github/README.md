# Eis Eck Ahrensbök

Webseite zum Anzeigen der tagesaktuellen Eissorten, geladen aus Google Sheets.

## Auf GitHub Pages veröffentlichen

1. Dieses Repository auf GitHub hochladen
2. In den Repository-Einstellungen → **Pages** → Source: **GitHub Actions** auswählen
3. Beim nächsten Push auf `main` wird die Seite automatisch gebaut und veröffentlicht

Die Adresse lautet dann: `https://DEIN-NAME.github.io/REPO-NAME/`

## Lokal entwickeln

```bash
npm install
npm run dev
```

## Eissorten bearbeiten

Die Eissorten kommen automatisch aus dem verknüpften Google Sheet (eine Sorte pro Zeile, erste Spalte).
